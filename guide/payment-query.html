Get a Transaction for Receiving Money

This API retrieves details of a specific transaction, including its status and payment details.

Request

HTTP Method & Endpoint

GET /api/v1/trades/:trade

HTTP Headers Header

KeyValueDescription
Acceptapplication/jsonSpecifies that the response format should be JSON.

Path parameters Path

KeyTypeRequiredSignedDescription
tradestringYesNoThe transaction ID provided by your system or our system.

Query parameters Query

KeyTypeRequiredSignedDescription
client_keystringYesYesThe API access key.
signaturestringYesNoSigned value for validation.

Request example

curl -X GET \
  https://example.com/api/v1/trades/20230101000000?client_key=01h6tn69wfcpy5q5x3vpb3x9me&signature=ba5df26991273c746960ce5238c6479e8ca6116381ac46cea96ffd30fafed082 \
  -H "Accept: application/json"

Response

Response parameters

KeyTypeDescription
client_keystringThe API access key.
amountstringThe transaction amount received.
trade_nostringThe transaction ID provided by our system.
out_trade_nostringThe transaction ID provided by your system.
payment_urlstringThe URL of the cashier payment page.
created_atstringThe transaction creation time in UTC±00:00.
paid_atstringThe transaction payment time in UTC±00:00.
statusintegerThe transaction status.

Transaction Status Codes

Status CodeDescription
0Unpaid
1Paid
2Expired

Response example

{
  "client_key": "01h6tn69wfcpy5q5x3vpb3x9me",
  "amount": "100.00",
  "trade_no": "100000012023072123389872",
  "out_trade_no": "1698896652712",
  "created_at": "2023-01-01T01:01:01.000000Z",
  "paid_at": "2023-01-01T01:02:03.000000Z",
  "status": 1
}
