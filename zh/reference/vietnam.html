指定银行 (可选)

如果您想指定银行代收，您可以在请求中包含字段extra.bank_code，不推荐。

extra.bank_code

银行编码银行名称
BIDVBIDV
VCBVietcombank
TCBTechcombank
MBBMB Bank

指定收银台页面 (可选)

默认显示所有信息，如果您希望向客户显示不同的收银台页面，您可以在请求中包含字段extra.type，不推荐。

extra.type

类型描述
1只显示银行信息
2只显示二维码
curl -X POST \
  https://example.com/api/v1/trades \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "client_key": "01h6tn69wfcpy5q5x3vpb3x9me",
    "amount": "10000",
    "channel_id": "1001",
    "out_trade_no": "20230101000000",
    "notify_url": "https://your-domain.com/webhook",
    "extra": "{\"bank_code\":\"VCB\", \"type\":\"1\"}",
    "signature": "ba5df26991273c746960ce5238c6479e8ca6116381ac46cea96ffd30fafed082"
  }'

业务参数 extra

如果您需要代付至越南的银行， extra.bank_code 字段为必填。 如果您需要代付至 Zalo 或 MoMo，请不要在您的请求中包含 extra 字段。

参数必填说明
bank_code是银行编码

NOTE

extra 必须是有效的 JSON 字符串。
